# Bidrag-Samhandler

## Kort beskrivelse
Bidrag-Samhandler håndterer informasjon om bidragssamhandlere og oppretter eller endrer samhandlere i aktørregisteret.

## Nøkkelinformasjon
| Felt | Verdi |
| --- | --- |
| Navn | Bidrag-Samhandler |
| Applikasjonstype | Backend-tjeneste |
| Driftsområde | GCP |
| GitHub | [navikt/bidrag-samhandler](https://github.com/navikt/bidrag-samhandler/) |
| Har database | Ikke dokumentert ennå |
| Eierteam | Bidrag |

## Viktige integrasjoner
- Brukes av `Bisys` når samhandlere opprettes eller endres
- Oppdaterer `Bidrag-Aktørregister`

## Databaser og lagring
Det er ikke dokumentert her om applikasjonen har egen database eller annen vedvarende lagring.

## Notater
Legg gjerne til informasjon om hvilke identifikatorer og datakilder som ligger til grunn for samhandlerinformasjonen.
