# Bidrag-bidragskalkulator-api

## Kort beskrivelse
Bidragskalkulator API beregner hvor mye en forelder skal betale eller motta i barnebidrag.
Applikasjonen fungerer som backend for bidrag-bidragskalkulator-ui.
