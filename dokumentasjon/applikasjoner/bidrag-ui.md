# Bidrag-ui

## Kort beskrivelse
Bidrag UI er hovedgrensesnittet til Bidrag. 
Bidrag UI er bare et skall grensesnitt som håndterer brukerinnlogging, sesjonshåndtering, serverlogging og metrikker.
